## Learnings

- Sorting
  - BubbleSort O(n^2)
  - InsertionSort O(n^2)
  - QuickSort O(n^2)
  - MergeSort O(nlogn)
- Searching
  - LinearSearch O(n)
  - BinarySearch O(logn)
  - Recursive BinarySearch O(logn)
- Math
  - Fibonacci O(n)
  - Recursive Fibonacci O(2^n)
  - Factorial O(n)
  - Recursive Factorial O(n)
  - isPrime O(sqrt(n))
  - isPowerOfTwo O(logn)
  - isPowerOfTwoBitWise O(1)
- Misc
  - CartesianProduct O(mn)
  - ClimbingStairCase O(n)
  - TowerOfHanoi O(2^n)
